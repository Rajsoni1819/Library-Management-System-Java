📚 Library Management System (Java)

📖 Project Overview

The Library Management System is a simple Java console application developed using Object-Oriented Programming (OOP). It allows users to manage library books through a menu-driven interface. Users can add, view, search, issue, return, and remove books. The project also uses file handling to save book records permanently.

---

🚀 Features

- Add a new book
- View all books
- Search book by ID
- Issue a book
- Return a book
- Remove a book
- Save data using File Handling
- Load saved books automatically when the program starts

▶️ How to Run

1. Open the project in VS Code.
2. Open the terminal inside the project folder.
3. Compile all Java files:

javac *.java

4. Run the project:

java Main
